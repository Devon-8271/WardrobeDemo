# 附录 — 当前模块一览

| 模块 | 职责 |
|---|---|
| `app.py` | FastAPI 入口，所有 HTTP 路由，会话管理，定时任务 |
| `db.py` | SQLite CRUD（wardrobe / user_profile / look） |
| `fashion_router.py` | 意图分类（Groq LLM → 规则兜底） |
| `fashion_dispatch.py` | 调度层，路由到对应处理函数 |
| `outfit_recommender.py` | 两阶段召回推荐，内存+磁盘双层缓存 |
| `outfit_generator.py` | 推荐图生成（调 tryon_skill） |
| `tryon_skill.py` | 统一试穿 Skill（单件/套装/拼图，含 pose_engine 集成） |
| `pose_engine.py` | 根据服装风格生成姿势描述 |
| `phase1_wardrobe.py` | Groq Vision 单品识别 |
| `phase2_tryon.py` | CLI 单件试穿（调 tryon_skill） |
| `look_manager.py` | 保存穿搭日志，触发 style_identity 更新 |
| `style_identity.py` | 从 look 历史推断用户风格标签 |
| `image2_client.py` | image2 图像生成客户端（submit/poll/download） |
| `mcp_label.py` | MCP server 批量打标（白底图 + 结构化标签） |
| `import_batch.py` | 批量导入 items.json 入库 |
